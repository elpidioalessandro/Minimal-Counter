# Minimal Counter App
## Descrizione
Minimal Counter è un'applicazione sviluppata in puro JavaScript che implementa un contatore interattivo.
L’interfaccia viene generata dinamicamente tramite la manipolazione del DOM e consente all’utente di incrementare e decrementare un valore numerico con un’esperienza fluida e minimale.

## Funzionalità
- Incremento e decremento del contatore tramite pulsanti
- Pressione lunga per variazione continua del valore
- Persistenza del valore tramite localStorage
- Personalizzazione dello step di incremento/decremento
- Animazione del numero per feedback visivo
- Controllo tramite tastiera (+ / - / r)

## Tecnologie usate
- JavaScript (Vanilla)
- HTML5
- CSS3 (iniettato via JavaScript)
- Web APIs (DOM manipulation, localStorage)

## Installazione e avvio

1. Copia l'URL web dal menu 'code'

2. Crea e apri una cartella denominata 'Minimal Counter' sul dekstop

3. All’interno di VS Code:
   - Clicca su Clona Git Repository
   - Seleziona la cartella 'Minimal Counter'
   - Apri il file `index.html`

4. Avvia il progetto:
   - Click destro su `index.html`
   - Seleziona **"Open with Live Server"** (se hai l’estensione installata)

   oppure

   - Fai doppio click su `index.html` per aprirlo direttamente nel browser.
